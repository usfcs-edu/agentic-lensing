# Phase 3a Reproduction: Huang et al. 2020 (arXiv:1906.00970)

**Paper:** "Finding Strong Gravitational Lenses in the DESI DECam Legacy Survey" (Huang, Storfer, Ravi, et al. 2020, ApJ 894 78). Uses the Lanusse 2018 *CMU DeepLens* ResNet-46 architecture (arXiv:1703.02642) re-implemented in TensorFlow; we re-implemented the same in PyTorch (script 01) following Lanusse §3.1 Fig 4.

**Reproduction scope:** Phase 3a = train the network from paper + public code only (NO Huang training code). Uses the NeuraLens-released candidate catalog as positives + DESI DR1 zcat-sampled galaxies as negatives. The full DECaLS deployment (sweep ~10⁶ galaxies → rank by ResNet score → human VI of top scores) is Phase 3b and explicitly out of scope here.

## 1. Training set

| Source | Count |
|---|---:|
| NeuraLens L18 lens candidates (positives) | 949 |
| DR1 zcat galaxies in DECaLS footprint (negatives) | 5,000 |

Split (70:20:10 train/val/test, stratified by label):

```
label     0    1
split           
test    490   93
train  3436  650
val     982  186
```

Cutout spec: 101×101 px grz at 0.262″/px ≈ 26.5″ FoV, `ls-dr9` layer, from `legacysurvey.org/viewer/fits-cutout`. Same size as Huang+2020 §3.2.

**Caveats versus Huang+2020's *actual* training set:**
- They used 613 known lenses from Master Lens Database + 6 specific recent publications (Carrasco, Diehl, Pourrahmani, Sonnenfeld, Wong, Jacobs); we use the NeuraLens-published L18 candidate list (949 systems) instead, which is their *output* and therefore includes some of their *input* training examples plus the newly discovered candidates.
- They used 13,000 non-lens cutouts in DECaLS with curated by-eye hard negatives (spirals, galaxy groups, cosmic rays, artifacts); we use 5,000 uncurated DR1 galaxies. Higher false-positive rate during deployment is expected.
- 5.3:1 class imbalance vs Huang+2020's 21:1.

## 2. Architecture

Lanusse 2018 / CMU DeepLens ResNet-46 (1 stem conv + 5 stages × 3 pre-activated bottleneck blocks). 3.5 M parameters at 3-channel input.

Per-stage spatial resolution for 101×101 input: 101 → 51 → 26 → 13 → 7 → AvgPool(1) → FC(512→1) → Sigmoid.

## 3. Hyperparameters (Lanusse §3.4 + Huang §3.2)

- Optimizer: **Adam**, default β
- Initial LR: **1e-3**, divided by 10 every 40 epochs
- Batch size: **128**
- Epochs: **120**
- Loss: **BCE with logits** (numerically stable)
- Preprocessing: per-band mean subtraction + std normalisation, clip ±250σ
- Augmentation: random ±90° rotation + flip + zoom [0.9, 1.0]

## 4. Results

- Best validation AUC: **0.9983** (at epoch 93)
- Final epoch train loss: 0.0178
- Final epoch val loss: 0.0849
- Total training wall-clock: 24.8 min
- **Held-out test AUC: 0.9991** (n_test=583, checkpoint epoch 93)

Compare to Huang+2020 published validation AUC = 0.98 (their 70:20:10 split on the 613-positive / 13,000-negative set on NERSC Cori).

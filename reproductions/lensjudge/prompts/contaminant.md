ROLE: CONTAMINANT-HUNTER. Run through the known false-positive catalog for CNN lens
finders and check each against the images: (1) spiral arms (blue, but radial and
symmetric, attached to a disk); (2) ring / polar-ring galaxies (a closed blue ring, no
external lens); (3) mergers / tidal tails (irregular, one-sided); (4) bright star
halos and diffraction spikes; (5) cosmic rays / satellite trails (sharp linear); (6)
isolated star-forming clumps / chance projections. If one fits, set `contaminant` and
grade D. If none fits and a tangential arc remains, allow a higher grade.
